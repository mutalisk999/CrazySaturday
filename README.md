# 疯狂星期六抢1大赛 🎱

一个基于 Python Tkinter 的桌球比赛模拟系统，支持选手管理、比赛进程跟踪、连胜统计等功能。

## 功能特性 ✨

### 核心功能
- **选手管理**：支持添加、编辑、删除选手，设置初始 HP（2-8 条命）
- **固定参赛选手**：管理常用选手信息（姓名、联系方式、邮箱、住址）
- **比赛模拟**：自动分配选手到球台，支持 1-9 张球台
- **HP 跟踪**：实时显示选手剩余命数，红色标记仅剩 1 条命的选手
- **连胜统计**：实时统计并显示全场最长连胜数
- **比赛记录**：记录每场比赛的胜负详情
- **逆天改命**：支持手动修改选手的生命值

### 界面特性
- 🎨 现代化 UI 设计，卡片式布局
- 📊 Treeview 表格展示选手信息
- 🔄 实时更新比赛状态
- 💾 支持保存/加载比赛状态
- ⏪ 支持状态回退和前进

## 快速开始 🚀

### 运行程序
```bash
python crazy_saturday.py
```

### 基本使用流程

1. **选手设置**
   - 添加参赛选手（姓名 + 初始 HP）
   - 或使用"固定参赛选手"功能快速导入

2. **开始比赛**
   - 点击"重新开始比赛"
   - 选择初始使用的球桌数量（1-9张）
   - 系统自动分配选手到球台

3. **比赛进行**
   - 点击"判负"按钮记录比赛结果
   - 系统自动处理人员轮转
   - 实时显示连胜统计和比赛记录

4. **撤台操作**
   - 点击球台上的"撤台"按钮标记该球台
   - 当前对局结束后该球台关闭
   - 选手移至场外候补区

5. **逆天改命**
   - 点击"逆天改命"按钮
   - 选择要修改的选手
   - 设置新的生命值

6. **数据管理**
   - 可随时保存/加载比赛状态
   - 支持状态回退和前进

## 比赛规则 📜

### 球台使用顺序
| 初始球台数 | 使用球台编号 |
|-----------|-------------|
| 1张       | 2号         |
| 2张       | 2-3号       |
| 3张       | 2-4号       |
| 4张       | 2-5号       |
| 5张       | 2-6号       |
| 6张       | 2-7号       |
| 7张       | 2-8号       |
| 8张       | 2-9号       |
| 9张       | 2-9号、1号  |

### 撤台规则
- 撤台由用户手动触发，点击球台上的"撤台"按钮
- 点击后该球台标记为"即将撤桌"状态（红色标题栏）
- 当前对局结束后，该球台关闭
- 球台上的选手移至场外候补区

### 比赛流程
1. 擂主 vs 挑战者，1局定胜负
2. 败方：移至场外候补区队尾，HP-1
3. 胜方：留在擂主位置
4. 候补区选手依次补位
5. HP=0 时移至淘汰区

### 特殊功能
- **逆天改命**：可修改未淘汰选手的生命值，用于纠正误操作或特殊情况处理
