# 疯狂星期六抢1大赛 🎱

一个基于 Python Tkinter 的桌球比赛模拟系统，支持选手管理、比赛进程跟踪、连胜统计等功能。

## 功能特性 ✨

### 核心功能
- **选手管理**：支持添加、编辑、删除选手，设置初始 HP（2-8 条命）
- **固定参赛选手**：管理常用选手信息（姓名、联系方式、邮箱、住址）
- **比赛模拟**：自动分配选手到球台，支持 1-9 张球台动态调整
- **HP 跟踪**：实时显示选手剩余命数，红色标记仅剩 1 条命的选手
- **连胜统计**：实时统计并显示全场最长连胜数
- **比赛记录**：记录每场比赛的胜负详情

### 界面特性
- 🎨 现代化 UI 设计，卡片式布局
- 📊 Treeview 表格展示选手信息
- 🔄 实时更新比赛状态
- 💾 支持保存/加载比赛状态
- ⚙️ 可配置球桌人数阈值

## 快速开始 🚀

### 运行程序
```bash
python crazy_saturday.py
```

### 基本使用流程

1. **选手设置**
   - 添加参赛选手（姓名 + 初始 HP）
   - 或使用"固定参赛选手"功能快速导入

2. **配置球桌阈值**（可选）
   - 点击"设定球桌阈值"
   - 修改各球桌触发人数
   - 或使用默认配置

3. **开始比赛**
   - 点击"重新开始比赛"
   - 系统自动分配选手到球台

4. **比赛进行**
   - 点击败方名字记录比赛结果
   - 系统自动处理人员轮转
   - 实时显示连胜统计和比赛记录

5. **数据管理**
   - 可随时保存/加载比赛状态
   - 支持历史记录回溯

## 比赛规则 📜

### 球台分配规则
| 参赛人数 | 球台数量 | 使用球台 |
|---------|---------|---------|
| 2-6人   | 1张     | 2号     |
| 7-10人  | 2张     | 2-3号   |
| 11-14人 | 3张     | 2-4号   |
| 15-17人 | 4张     | 2-5号   |
| 18-20人 | 5张     | 2-6号   |
| 21-24人 | 6张     | 2-7号   |
| 25-27人 | 7张     | 2-8号   |
| 28-31人 | 8张     | 2-9号   |
| ≥32人   | 9张     | 1-9号   |

### 减球台规则
- 减台顺序：1→9→8→7→6→5→4→3→2
- 当剩余选手达到阈值时自动减台
- 减台时选手移至场外候补区

### 比赛流程
1. 擂主 vs 挑战者，1局定胜负
2. 败方：移至场外候补区队尾，HP-1
3. 胜方：留在擂主位置
4. 候补区选手依次补位
5. HP=0 时移至淘汰区
